# 📘 Semana 2: JavaScript ES6+

Este proyecto forma parte del proceso de aprendizaje de **JavaScript moderno (ES6+)**, aplicando nuevos conceptos mediante ejercicios prácticos.

## Objetivos alcanzados

- [x] Uso de `let` y `const` para declaración de variables  
- [x] Implementación de **arrow functions**  
- [x] Aplicación de **destructuring** y operador **spread**  
- [x] Uso de **métodos de arreglos** (`map`, `filter`, `reduce`, etc.)  
- [x] Uso de **template literals** para interpolación de texto  

## Ejercicios completados
1. **Gestión de aprendices** (`ejercicios.js`)  
   - Manipulación de arreglos de objetos.  
   - Uso de métodos como `map()` y `filter()` para procesar información.

2. **Lista de tareas** (`lista-tareas.js`)  
   - Creación y gestión dinámica de tareas.  
   - Aplicación de funciones flecha y métodos de arreglo.

3. **Sistema de contactos** (`contactos.js`)  
   - Gestión de información mediante objetos y arreglos.  
   - Uso de destructuring y spread para optimizar el código.

## Aprendizajes clave
Durante esta semana comprendí la importancia de escribir código más limpio y eficiente utilizando las nuevas características de ES6+.  
El uso de `let` y `const` mejora la seguridad del código, las arrow functions permiten una sintaxis más clara, y los métodos de arreglos facilitan el trabajo con colecciones de datos.  
El destructuring y el operador spread ayudan a simplificar la manipulación de objetos y arreglos.

## Dificultades superadas
Uno de los principales retos fue comprender correctamente el funcionamiento de los métodos de arreglos, especialmente `reduce()`. También fue un desafío entender cuándo usar `let` y cuándo `const`, así como la diferencia entre funciones tradicionales y funciones flecha. Con práctica y ejemplos aplicados en los ejercicios, logré reforzar estos conceptos y aplicarlos correctamente.
